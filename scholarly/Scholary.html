Descriere

Aplicatia de fata este construita cu scopul de a invata, in mod interactiv , cele mai inportante concepte legate de Tehnologiile Web.

In cadrul acestui proiect se vor folosi si dezvolta concepte precum : protocolul http , cookie-uri , sesiuni Web, server Web , servicii Web - care includ la randul lor conceptul paradigmei REST , transferul asincron de pachete folosind Ajax , mash-up-uri si alte concepte care stau la baza tehnologiilor web.

Aplicatia web este reprezentata de un joc online in care utilizatorul se aventureaza in lumea cibernetica cu scopul de a-si crea un webpage cat mai complex. El va invata trecand de fiecare nivel cate un concept legat de telul sau - crearea propriului lui workspace online.

In acelasi timp , utilizatorul va putea interactiona si cu alti jucatori prin: invitarea unui prieten in a-l ajuta la trecerea nivelului, achizitionarea de diferite item-uri din market-ul jocului.

Tehnologii utilizate

HTML & CSS - pentru a crea interfata paginii principale

MongoDB - pentru mentinerea stadiilor unui cont

Bootstrap 3 - pentru implementarea functionalitatilor vizuale - butoane/functii/imagini/etc.

Javascript - pentru dezvoltarea aplicatiei din partea serverului si clientului

Phaser - pentru dezvoltarea jocului - crearea de harti , animatii ,caractere ,etc.
